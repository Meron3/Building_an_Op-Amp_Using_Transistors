# Building an Op-Amp Using Transistors

This is a KiCad project for a speaker amplifier using 16 transistors.

It can be opened with KiCad 9.0 or later.

The toggle switch footprint uses KiCad-lib imported by nosuz.
https://github.com/nosuz/kicad-lib
